## O que é uma *Table*?

Uma **tabela** é formada por **colunas (*columns*)**, e cada coluna contém **linhas (*rows*)**.

Normalmente, uma tabela possui uma coluna especial chamada **ID**, que funciona como um **identificador único**, também conhecida como **Primary Key**.

Cada **row** representa um registro e contém os **data values** (dados) daquela entrada.
