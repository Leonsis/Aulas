# USE e SELECT
Usando o comando `USE` é possivel selecionar o banco de dados no qual vai ser realizada as Querys
`USE sakila`;

# SELECT e FROM
`SELECT actor_id, first_name FROM actor;`
Com o select é possivel selecionar uma Coluna da Tablea informada depois do comando `FROM`
Com o seguinte comando é posivel selecionar todas as colunas da tabela informada `SELECT * FROM actor`.
